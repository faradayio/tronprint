# Quick start: using Tronprint with a Rails app

In your application's Gemfile:

<figure>
  <figcaption>Gemfile</figcaption>
{% highlight ruby %}
gem 'tronprint'
{% endhighlight %}
</figure>

Followed by:

<figure>
  <figcaption>Shell</figcaption>
{% highlight console %}
$ bundle install
{% endhighlight %}
</figure>
